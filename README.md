## 📂 Структура проекта

```plaintext
📂 src/
├── 📄 __init__.py             # Инициализация пакета
├── 📄 main.py                 # Точка входа в приложение
├── 📄 app.py                  # Главный класс приложения (App)
├── 📄 utils.py                # Вспомогательные функции (центрирование окон и т.д.)
├── 📂 screens/                # Директория с классами экранов
│   ├── 📄 __init__.py         # Инициализация пакета screens
│   ├── 📄 base_screen.py      # Базовый класс для всех экранов
│   ├── 📄 login_screen.py     # Экран авторизации
│   ├── 📄 main_screen.py      # Главный экран
│   └── 📂 main_screen_panels/ # Компоненты главного экрана
│       ├── 📄 __init__.py     # Директория с панелями экрана main_screen
│       ├── 📄 top_panel.py    # Верхняя панель
│       ├── 📄 left_panel.py   # Левая панель
│       ├── 📄 right_panel.py  # Правая панель
│       └── 📄 central_panel.py# Центральная панель
├── 📂 assets/                 # Ресурсы приложения
│   └── 📜 logo.png            # Логотип приложения
├── 📄 requirements.txt        # Список зависимостей
└── 📄 README.md               # Документация проекта
